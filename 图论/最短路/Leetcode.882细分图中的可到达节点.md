### 数据范围
* $0 <= edges.length <= min(n * (n - 1) / 2, 10^4)$
* $edges[i].length == 3$
* $0 <= ui < vi < n$
* $图中 不存在平行边$
* $0 <= cnti <= 10^4$
* $0 <= maxMoves <= 10^9$
* $1 <= n <= 3000$
